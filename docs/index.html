The procedure

Three entries on the docket.

ENTRY 01
Drop
Drag in audio files, or an entire folder. StenoDrop searches every subfolder and queues each audio file it finds.
ENTRY 02
Transcribe
Speech recognition runs right on your machine, accelerated by Apple Silicon. A five-second clip finishes in under a second, and nothing ever touches the network.
ENTRY 03
Read
Each transcript is saved as name.txt beside its audio file, and kept in the app so you can copy it or reveal it in Finder.
The exhibits

A stenographer that lives on your machine.

Cloud services charge by the minute and read your audio on their servers. StenoDrop does the same work locally, for nothing.

Exhibit A: Private

Nothing leaves your machine

Your audio and transcripts are processed and stored locally, every time. Airplane mode works just fine.

Exhibit B: Batch

Folders, not files

Point it at a year of voice memos. StenoDrop queues everything, works through the list, and skips duplicates automatically.

Exhibit C: اردو

Urdu → English, built in

First-class Urdu support with one-tap translation to English, or keep the original script. 100 languages in total.

Exhibit D: Formats

Any audio you have

wav, mp3, m4a, flac, opus, voice memos, even the audio from your videos.

Exhibit E: Free

$0, no strings

Open source. No account, no trial, no meter. The voice-recognition software downloads once (about 466 MB) and it's yours to keep.

Exhibit F: Fast

Hardware-accelerated

Batches that would cost hours of upload time finish while you make tea.

Speak in any language. Read it in English.

One toggle turns Urdu, Hindi, Arabic, Punjabi, Pashto, or any of 100 languages, into clean English text. Switch translation off to keep the original script, or set the language manually when a short clip confuses auto-detect.

Appendix A: macOS prerequisites
engine install
# free engine + audio converter
brew install whisper-cpp ffmpeg

# unsigned beta, clear quarantine once
xattr -cr /Applications/StenoDrop.app

First launch offers a one-click download of the voice-recognition software (about 466 MB). After that, you're fully offline. Windows and Linux builds bundle everything already, so there's nothing extra to install.

Appendix B: build from source
any platform
git clone https://github.com/fahdi/stenodrop.git
# macOS (Swift):
cd stenodrop/mac && ./scripts/make-app.sh
# Windows / Linux (Tauri + Rust):
cd stenodrop/desktop && cargo tauri build

Open source. Audit it, fork it, or send a pull request. The repository has build docs for every platform.
