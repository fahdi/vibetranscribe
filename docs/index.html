The procedure

Three entries on the docket.

ENTRY 01
Drop
Drag audio files — or a whole folder — onto the window. StenoDrop searches subfolders and queues every audio file it finds.
ENTRY 02
Transcribe
Whisper runs on your machine — Metal-accelerated on Apple silicon. A five-second clip takes under a second. Nothing touches the network.
ENTRY 03
Read
Each transcript is entered as name.txt beside its audio file — and kept in the app to copy or reveal in Finder.
The exhibits

A stenographer that lives on your machine.

Cloud services charge by the minute and read your audio on their servers. StenoDrop does the same work locally, for nothing.

Exhibit A — Private

Nothing leaves your machine

Audio, transcripts, everything — processed and stored locally. Airplane mode works fine.

Exhibit B — Batch

Folders, not files

Point it at a year of voice memos. It queues everything, works through the list, skips duplicates.

Exhibit C — اردو

Urdu → English, built in

First-class Urdu with one-toggle translation to English — or keep the original script. 100 languages total.

Exhibit D — Formats

Any audio you have

wav, mp3, m4a, flac, opus, voice memos — even the audio track of videos.

Exhibit E — Free

$0, no strings

Open source. No account, no trial, no meter. The Whisper model downloads once (~466 MB) and it's yours.

Exhibit F — Fast

Hardware-accelerated

Batches that would cost hours of upload time finish while you make tea.

Speak in any language. Read it in English.

Whisper's translate mode turns Urdu, Hindi, Arabic, Punjabi, Pashto — any of 100 languages — into clean English text. Or switch translation off and keep the original script. Force a specific language when short clips fool auto-detection.

Appendix A — macOS prerequisites
engine install
# free engine + audio converter
brew install whisper-cpp ffmpeg

# unsigned beta — clear quarantine once
xattr -cr /Applications/StenoDrop.app

First launch offers a one-click Whisper model download (~466 MB). After that you're fully offline. Windows & Linux builds bundle the engine — no prerequisites.

Appendix B — build from source
any platform
git clone https://github.com/fahdi/stenodrop.git
# macOS (Swift):
cd stenodrop/mac && ./scripts/make-app.sh
# Windows / Linux (Tauri + Rust):
cd stenodrop/desktop && cargo tauri build

Open source — audit it, fork it, or send a PR. The repository has per-platform build docs.
